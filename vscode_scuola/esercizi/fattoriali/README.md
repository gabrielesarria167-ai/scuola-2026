Questo esercizio prende un numero intero dall'utente e ritorna il fattoriale di esso

La funzione dovrebbe:

- Ricevere il numero
- Trovare il fattoriale rendendo la funzione ricorsiva

Un fattoriale è un numero n che viene moltiplicato n-1 volte partendo dal numero e diminuendo di 1 ogni volta fino ad arrivare ad 1

eg. 5! = 5 * 4 * 3 * 2 * 1

La funzione dovrebbe essere circa cosi:

```cpp

int fattoriale(numero base){
    return base-1
}

```
